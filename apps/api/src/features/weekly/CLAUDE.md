# features/weekly (api)

## Purpose
Weekly reflection ritual — Sunday/Monday check-in flow. Five questions (rating, win, intention, calendar preview, AI intention). Results feed into Monday's routine generation.

## Patterns & Rules
- `weekStartDate` is always the Monday of the current week in ISO format ("yyyy-MM-dd")
- Unique index: `{ userId: 1, weekStartDate: 1 }` — one reflection per user per week
- AI intention generated by `generateAIIntention()` — medical data, medicines, personal life answers never in prompt
- If AI fails or is unavailable → graceful fallback sentence returned, never an error to the user
- `runWeeklyReflectionNotifications()` runs hourly via node-cron — checks per-user timezone for Sunday 7pm / Monday 7am windows
- Monday routine injection: `routine.service.ts` fetches previous week's reflection for Monday generation
