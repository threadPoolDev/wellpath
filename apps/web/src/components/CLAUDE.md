# apps/web/src/components

## Purpose
Shared UI components used across multiple features. Not feature-specific.

## Patterns & Rules
- Only truly shared components live here (e.g. Avatar, used in nav, groups, profile)
- Feature-specific components live inside `features/<name>/components/`
- All shadcn primitives generated by CLI go in `components/ui/`

## Contents
| File/Folder | Purpose |
|---|---|
| Avatar.tsx | Photo + initials fallback — color derived from name hash, consistent across renders |
| ui/ | shadcn/ui generated primitives |

## Avatar usage
Props: `name`, `photoUrl?`, `thumbnailUrl?`, `size: sm|md|lg`, `variant: full|thumbnail`
- `variant="thumbnail"` prefers `thumbnailUrl` (faster); `variant="full"` uses `photoUrl`
- Color is deterministic from name hash — never random, so it's stable across page loads
